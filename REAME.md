## Описание программы
Архитектура и логика работы:
Интерфейс (Tkinter): Используется встроенная библиотека Python для создания графического окна. Оно содержит поля ввода для названия, жанра, года выпуска и рейтинга, а также кнопку «Добавить» и таблицу для просмотра списка.
Хранение данных: Информация сохраняется в текстовый файл movies.json в формате JSON. Это позволяет данным не пропадать после закрытия программы.
Обработка ошибок: В коде реализована проверка (валидация). Если пользователь введет текст вместо года или укажет некорректный рейтинг (вне диапазона 0–10), программа выдаст предупреждение и не сохранит запись.
Автоматизация: При запуске приложение автоматически проверяет наличие файла базы данных, считывает его и заполняет таблицу актуальными записями.
Структура: Код написан с использованием классов (ООП), что делает его легко читаемым и удобным для расширения (например, если вы захотите добавить кнопку удаления).
## Требования для запуска

Python 3.6+
Библиотеки: 'tkinter' (встроен в Python), 'json' (встроен в  Python)
